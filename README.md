# Flask_Todo_app

## Setup:

### POSIX:
```bash
python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt

export FLASK_APP=app.py

flask run
```

### Windows:
```cmd
python3 -m venv venv

venv/Scripts/activate.bat

pip install -r requirements.txt

set FLASK_APP=app.py

flask run
```
